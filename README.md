
# 📊 Crypto Analytic Dashboard

## 📌 Descrierea proiectului

**Crypto Analytic Dashboard** este o aplicație web care permite utilizatorilor să urmărească piața criptomonedelor în timp real. Utilizatorii pot vizualiza prețuri, grafice, trenduri și informații esențiale despre diverse criptomonede.

Aplicația utilizează API-uri externe pentru a prelua date actualizate și pentru a le afișa într-un mod interactiv și ușor de înțeles.

Scopul proiectului este de a demonstra utilizarea:

* HTML
* CSS
* JavaScript
* API-uri externe (crypto market data)
* Vizualizare date (charts)

---

## 🚀 Cum se deschide proiectul

1. Descarcă sau clonează repository-ul de pe GitHub:

```bash
git clone https://github.com/nume-user/Crypto-Analytic-Dashboard.git
```

2. Intră în folderul proiectului:

```bash
cd Crypto-Analytic-Dashboard
```

3. Deschide fișierul `index.html` într-un browser
   **sau** deschide proiectul în Visual Studio Code.

4. Introdu numele unei criptomonede (ex: Bitcoin, Ethereum) în bara de căutare pentru a vedea:

* prețul actual
* evoluția (chart)
* capitalizarea de piață
* volum
* alte statistici relevante

---

## 🛠 Tehnologii folosite

* HTML
* CSS
* JavaScript
* API pentru date crypto (ex: CoinGecko / Binance / CoinMarketCap)
* Chart.js (pentru grafice)

---

## ⚡ Funcționalități principale

* 🔍 Căutare criptomonede
* 📈 Grafice live (price action)

* 📊 Market cap & volum
* 🌙 UI modern (dark/light mode — opțional)
